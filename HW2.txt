Параметры host и port нужно гуглить, в зависимости от используемого вами почтового сервиса. 
Гуглится по запросу: SMTP название_почтового_сервиса
Пример: 
https://support.google.com/mail/answer/7126229?hl=ru
или
https://help.mail.ru/mail/mailer/popsmtp

ВАЖНО!!! Личные данные для логина в почтовом сервисе вынести в отдельный файл и использовать их только для подключения в главном
Не забудьте добавить его в .gitignore, чтобы ваши личные данные не попали в github
